    1. Install KLayout

     

    KLayout is a free dxf and gds viewer and editor.

    The save function of ShapelyChipDesigns is based on KLayout. Install Klayout from here: http://www.klayout.de/

    As Klayout will be called in the command line it has to be available in the cmd window.

    For this the klayout path needs to be added to the path variable, like this:

      1. On my computer the path to the klayout application is:

      C:\Users\Anna\AppData\Roaming\KLayout (64bit)

      You can find it quickly by opening the file location of the application:

       
      1. Then right click on "Computer">choose "Properties">"Advanced system settings">"Environment variables"

       

      1. Add the klayout path from step A). Watch out to place the semicolon before the klayout path but no space. Do not add a semicolon behind the path:

       
      1. Test by typing klayout in a cmd-command prompt:

       

       

      This should open now the klayout application
